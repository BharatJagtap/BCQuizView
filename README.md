# BCQuizView
<p>
BCQuizView contains a set of classes that can be used to create quiz implementation in an iOS App. It supports below type of questions .  
<ul>
<li>Boolean </li>
<li>Single Select</li> 
<li>Multiple Select </li>
<li>Text Area </li>
<li>Dropdown </li>
</ul>
